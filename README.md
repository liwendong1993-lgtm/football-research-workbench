# 足球研究工作台 v1

一个零依赖、零服务器费用、本地优先的足球比赛研究工具。

## 最简单的启动方法（Mac）

双击：

`启动足球研究工作台.command`

浏览器会自动打开 `http://127.0.0.1:8766/`。请保持启动脚本的终端窗口开启。

如果 macOS 第一次阻止运行：右键脚本 → 打开 → 再次确认“打开”。

## 已实现

- 自动读取中国体彩网当期足球比赛、让球和参考赔率
- 日期切换与全部/已编辑/主推筛选
- 胜平负、让球胜平负、进球数、比分、信心和文字分析
- 本地自动保存，刷新页面后仍保留
- 多场方案组合与参考赔率自动计算
- 生成1080像素宽的微信长图并保存PNG
- 保存历史报告，历史报告可重新生成图片
- 手动添加比赛，接口失效时仍可工作
- JSON完整备份和恢复
- PWA离线缓存，可添加到手机主屏幕

## 数据保存位置

数据保存在当前浏览器的 Local Storage 中，不会上传到服务器。清理浏览器网站数据会删除记录，所以请定期到“设置 → 导出JSON”备份。

不同浏览器、不同电脑之间不会自动同步。可通过导出/导入JSON迁移。

## 手机使用

电脑与手机在同一个局域网时，可以把服务绑定到局域网地址后访问；第一版默认只绑定本机，避免无意公开数据。也可以将整个项目部署到免费静态托管平台。

## 注意

比赛与参考赔率来自中国体彩网公开页面接口，接口可能调整。正式商业使用前请确认数据授权。本工具不提供投注、代购、跟单或在线支付。
